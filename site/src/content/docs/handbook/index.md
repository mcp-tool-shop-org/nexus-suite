Welcome to the Nexus Suite handbook. This guide covers everything you need to install, configure, and operate the six composable Python packages that make up Nexus Suite.

## What you will find here

- **[Getting Started](/nexus-suite/handbook/getting-started/)** -- Clone the repo, install a component, and run your first test in under two minutes.
- **[Architecture](/nexus-suite/handbook/architecture/)** -- Three-layer design: control, routing, and attestation.
- **[Projects](/nexus-suite/handbook/projects/)** -- Deep dive into each package: nexus-router, nexus-attest, nexus-control, and the adapter family.
- **[Reference](/nexus-suite/handbook/reference/)** -- CLI flags, configuration options, and adapter API surface.

## Who this is for

Nexus Suite is built for teams that operate MCP tool ecosystems and need governance, cryptographic attestation, or request routing between tools. If you use MCP servers and want policy enforcement, signed outputs, or pluggable transport adapters, this handbook will get you there.
